Proton Accelerator Neutron Shielding Dataset and ML Analysis This repository contains datasets and Python codes used for curve fitting and machine learning (ML) analysis in our study on secondary neutron dose attenuation in various shielding environments for high-energy proton accelerators.

The datasets were generated from high-statistics FLUKA Monte Carlo simulations performed by Demet Sariyer. The machine learning models, statistical analyses, and Python implementations were developed by Ali Osman Gökcan.

**Machine learning models:**

Linear Regression (LR) Decision Tree Regressor (DTR) Random Forest Regressor (RFR) Evaluation metrics: R², RMSE, MAE, and 5-fold cross-validation

**Programming language:** Python 3.x
